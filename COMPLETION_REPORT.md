## 🚀 使用方法

### クイックスタート

1. **Google Apps Scriptにデプロイ**
   ```
   DEPLOY.md の手順に従ってください
   ```

2. **Google Docsで使用**
   - メニューから「🪽 ロンリちゃん」を選択
   - 「論理の飛躍を検出」をクリック

3. **設定のカスタマイズ**
   - メニュー→「設定」から調整可能
   - ランダム検出率：0-20%
   - AI検出：ON/OFF
   - 効果音：ON/OFF

### サンプルテキスト

**検出されるパターン:**
```
太陽は東から昇る。だから月は西から昇る。
当然、このことは誰もが知っている。
つまり、論理は完璧である。
```

**検出されないパターン:**
```
太陽は東から昇る。これは地球の自転によるものだ。
同様に、月も地球の周りを公転している。
したがって、月の満ち欠けが起こる。
```

### よくある問題と解決策

**Q: メニューに表示されない**
```
解決策: ページをリロード（F5）
```

**Q: エラー: "Script function not found"**
```
解決策: Code_Complete.gs を Code.gs にリネームして使用

**Q: アニメーションが動かない**
```

1. HTMLファイル名を確認（stamp_animation.html）

```

```
解決策:
